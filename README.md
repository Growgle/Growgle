# **Growgle – India’s First Democratised Geopolitical Career Compass**

### *Where Careers Meet Intelligence — Personalised, Real-Time, AI-Driven.*

---

## Overview

**Growgle** is India's first **pan-industrial, AI-driven career ecosystem** powered by Google’s latest AI models and cloud technologies.
It combines **career intelligence**, **live economy & policy insights**, **RAG-powered knowledge**, and **hyper-personalised recommendations** to build a unified career platform for every Indian — students, job-seekers, working professionals, founders, and policy aspirants.

Growgle integrates:

* Real-time **job intelligence**
* Government policy & geopolitics analysis
* Personalised **career roadmaps**
* AI-powered **mock interviews**
* **Multi-dimensional RAG** datasets (4.7M token vector DB)
* **Industry & skill trend dashboards**
* Automated **career task planning** (Calendar + Tasks)
* AI resume enhancer & cover-letter generator
* Voice-based interactions (Speech-to-text + Translation)

Growgle bridges the gap between **policy, industry, and people** — enabling a smarter, fairer, more democratized future of work.

# Key Features

### **1. Career Insights Engine**

* Fetches **live news**, industry signals, and policy updates
* Performs **trend analysis**, sentiment, conflict detection
* Generates personalised insights using **Gemini 2.5 + Gemma**
* Summarizes multi-source data into actionable advice

---

### **2. AI-Powered Job Discovery**

* Multi-platform job aggregation (Google Talent Solution + JSearch)
* Semantic ranking using **Vertex Matching Engine**
* “Opportunity Fit Score” generated using Gemini embeddings
* Filters by skills, experience, domain, work culture, location, and more

---

### **3. Personalised Career Roadmaps**

* Dynamic paths for:

  * Industry roles
  * UPSC / Govt roles
  * Higher studies (Masters, PhD)
  * Startup & entrepreneurship
* Auto-adjusts based on:

  * Industry movement
  * Policy changes
  * User skill graph
  * Regional opportunities

---

### **4. Growgle RAG Database (4.7M Token)**

* Largest hybrid knowledge-base for careers in India
* Includes:
  ✔ Government data (NITI, MeitY, MoE, NSDC, MoD…)
  ✔ Google Data Commons
  ✔ Industry reports & McKinsey insights
  ✔ Business outlook reports
  ✔ Labour market statistics
  ✔ Policy think-tank data
* Stored using **vector embeddings + BigQuery + Vertex Matching Engine**

---

### **5. AI Resume, Cover Letter & Skill Gap Analyzer**

* PDF → Text extraction
* Auto keyword comparison using embeddings
* ATS-optimized resume suggestions
* AI-generated cover letter
* Skill gap radar chart

---

### **6. Career Mentor Chatbot (Growgle Assistant)**

* Powered by **Gemini 2.5 Flash Pro / Gemma**
* Multilingual (using Translate API)
* Voice-based interactions (Speech-to-Text)
* Can fetch:

  * Job insights
  * Trend analysis
  * Policy updates
  * Roadmap suggestions
  * Resume improvements

---

### **7. Task & Calendar Automation**

* Uses Google Calendar + Tasks
* Auto-creates:

  * Study plans
  * Application deadlines
  * Roadmap milestones
* Smart planning using Gemini reasoning
